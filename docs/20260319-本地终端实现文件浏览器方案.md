# 本地终端实现文件浏览器方案

> 日期：2026-03-19
> 前置文档：[第二期方案](./20260319-桌面本地终端AI等能力扩展实现方案.md)
> 状态：设计中

---

## 1. 背景

当前文件浏览器插件（`builtin-file-browser`）所有操作通过 `window.electron.sshExecute()` 执行远程 Shell 命令实现。本地终端模式下 `connectionId` 为 null，文件浏览器完全不可用。

本方案目标：引入 `IFileBrowserService` 抽象接口，新增 `LocalFileBrowserService` 实现（通过 Main 进程 `fs` API 操作本地文件系统），让文件浏览器在本地终端模式下可用。

---

## 2. 设计原则

| 原则 | 说明 |
|------|------|
| **接口抽象** | 定义 `IFileBrowserService`，SSH 和本地各自独立实现 |
| **零改动 UI** | `FileBrowserPanel` 及其子组件不改动，仅切换服务实现 |
| **零改动 SSH** | 现有 `FileBrowserService`（SSH）不改动逻辑，仅实现接口 |
| **Main 进程执行** | 本地文件操作在 Main 进程通过 `fs` API 执行，Renderer 通过 IPC 调用 |
| **本地特化** | 上传/下载在本地模式下无意义（源和目标都是本机），隐藏相关按钮 |

---

### 3.2 路由机制

```
TerminalView 渲染底部面板
       │
       ▼
connectionId = host.connectionType === 'local' ? null : connectionId
       │
       ▼
FileBrowserPanel 收到 connectionId
       │
       ├── connectionId 非空 → new FileBrowserService(connectionId)   [SSH]
       └── connectionId 为空 → new LocalFileBrowserService()          [本地]
       │
       ▼
UI 调用 service.listFiles() / service.readFileForEdit() / ...
       │
       ├── SSH: sshExecute('ls -alh ...') → 解析输出
       └── Local: IPC 'local-fs-list' → Main 进程 fs.readdir+stat → FileItem[]
       │
       ▼
统一返回 FileItem[] / string / void → UI 渲染
```

**关键**：路由发生在 `FileBrowserPanel` 内部（根据 `connectionId` 是否为 null），不在 TerminalView。TerminalView 只负责传递 `connectionId`（本地模式传 null）。

### 3.3 本地模式特化

| 功能 | SSH 模式 | 本地模式 | 说明 |
|------|---------|---------|------|
| 列出文件 | `ls -alh` + 解析 | `fs.readdir` + `fs.stat` | 返回相同 FileItem[] |
| 读取文件 | `cat \| base64` | `fs.readFile` | 返回相同 string |
| 写入文件 | `echo \| base64 -d > file` | `fs.writeFile` | 相同接口 |
| 目录树 | `find -maxdepth` | 递归 `fs.readdir` | 返回相同 DirectoryNode[] |
| 删除 | `rm -rf` | `fs.rm({ recursive })` | 相同接口 |
| 重命名 | `mv` | `fs.rename` | 相同接口 |
| 权限 | `chmod` | `fs.chmod` | 相同接口 |
| 打包 | `tar czf` | `child_process.exec('tar')` | 相同接口 |
| **上传/下载** | SFTP 传输 | **不支持（隐藏按钮）** | 本地模式源和目标都是本机 |
| **sudo 写入** | `echo pwd \| sudo -S` | **不支持** | 安全考虑，不做提权 |
| **初始路径同步** | `sshPwd()` | `os.homedir()` | 本地默认打开 home 目录 |

## 4. 核心流程

### 4.1 本地文件列表加载

```
FileBrowserPanel 首次可见（connectionId === null）
  │
  ▼
创建 LocalFileBrowserService
  │
  ▼
初始路径 = os.homedir()（通过 IPC 获取）
  │
  ▼
service.listFiles(homedir)
  → IPC: 'local-fs-list'
  │
  ▼
Main 进程: LocalFsHandler.list(path)
  → fs.readdir(path, { withFileTypes: true })
  → 对每个 entry: fs.stat(fullPath)
  → 构造 FileItem { name, size, type, mtime, permission, userGroup, isDir }
  │
  ▼
返回 FileItem[] → FileBrowserPanel 渲染
```

### 4.2 本地文件编辑

```
用户双击文件
  │
  ▼
service.readFileForEdit(path)
  → IPC: 'local-fs-read'
  │
  ▼
Main 进程: LocalFsHandler.read(path)
  → 检查文件大小（>2MB 拒绝）
  → fs.readFile(path, 'utf-8')
  │
  ▼
返回 string → FileEditorModal 显示
  │
  ▼
用户编辑后保存
  │
  ▼
service.writeFileContent(path, content)
  → IPC: 'local-fs-write'
  │
  ▼
Main 进程: fs.writeFile(path, content, 'utf-8')
```

### 4.3 本地目录树加载

```
service.getDirectoryTree('/', 3)
  → IPC: 'local-fs-tree'
  │
  ▼
Main 进程: LocalFsHandler.tree(path, maxDepth)
  → 递归 fs.readdir
  → 仅保留目录（过滤文件）
  → 限制深度 ≤ maxDepth
  → 构造 DirectoryNode[]
  │
  ▼
返回 DirectoryNode[] → FileTreePanel 渲染
```

## 6. 重点问题

| 问题 | 解决方案 |
|------|---------|
| **上传/下载无意义** | 本地模式隐藏上传/下载按钮。在 `FileBrowserPanel` 中根据 `connectionId === null` 判断 |
| **sudo 写入** | 本地模式不支持 sudo 写入，编辑器的"sudo 保存"按钮在本地模式隐藏 |
| **权限不足** | `fs.readFile` / `fs.writeFile` 受当前用户权限限制，操作失败时友好提示 |
| **大文件** | 读取限制 2MB（与 SSH 一致），列表不受限 |
| **符号链接** | 使用 `fs.lstat` 检测链接类型，`fs.stat` 获取链接目标信息 |
| **隐藏文件** | `fs.readdir` 默认包含隐藏文件（与 SSH `ls -a` 一致） |
| **Windows 路径** | 使用 `path.join()` / `path.sep`，根目录为盘符（`C:\`） |
| **初始路径** | 本地模式默认 `os.homedir()`，不调 `sshPwd()` |
| **FileItem.type** | 本地模式沿用相同字段值：`'文件'`/`'文件夹'`/`'链接'`/`'可执行文件'` |
| **性能** | `fs.readdir` + `stat` 对大目录可能慢，限制单次最多返回 1000 项 |
| **打包功能** | 本地 `tar` 通过 `child_process.exec` 执行，与 SSH 行为一致 |

## 7. 实现优先级

| 优先级 | 模块 | 说明 |
|--------|------|------|
| P0 | `IFileBrowserService` 接口 | 抽象定义 |
| P0 | `LocalFsHandler`（Main 进程） | 核心 fs 操作 |
| P0 | Preload `localFs` IPC 通道 | 通信桥梁 |
| P1 | `LocalFileBrowserService` | Renderer 侧实现 |
| P1 | `FileBrowserPanel` 路由 | 根据 connectionId 创建服务 |
| P2 | 隐藏上传/下载/sudo 按钮 | 本地模式特化 UI |
| P2 | 初始路径默认 homedir | 替代 sshPwd |
