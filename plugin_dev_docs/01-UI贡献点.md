# TermCat 插件 UI 贡献点

> 本文档描述 TermCat 插件系统提供的所有 UI 扩展点，包括内置插件和外部插件两种模式。

---

## 概述

TermCat 插件系统提供两类 UI 贡献机制：

| 类型 | 运行环境 | UI 注册方式 | 适用场景 |
|------|---------|------------|---------|
| **内置插件** | Renderer 进程 | 直接注册 React 组件 | 应用自带的功能模块 |
| **外部插件** | Main 进程 | 模板驱动（JSON 描述 → 预定义模板渲染） | 第三方扩展 |

---

## 1. 侧栏面板（Sidebar Panel）

### 内置插件方式

通过 `context.registerSidebarPanel()` 注册 React 组件到左侧或右侧边栏。

```typescript
import type { BuiltinPluginContext, SidebarPanelRegistration } from '../types';

context.registerSidebarPanel({
  id: 'my-sidebar',
  position: 'right',          // 'left' | 'right'
  component: MySidebarPanel,   // React 组件
  defaultWidth: 320,
  defaultVisible: false,
  storageKeyPrefix: 'my-plugin',  // localStorage 持久化前缀
});
```

**组件接收的 Props（`SidebarPanelProps`）：**

| Prop | 类型 | 说明 |
|------|------|------|
| `sessionId` | `string` | 当前终端会话 ID |
| `connectionId` | `string` | SSH 连接 ID |
| `connectionType` | `'ssh' \| 'local'` | 连接类型 |
| `width` | `number` | 面板宽度 |
| `isVisible` | `boolean` | 是否可见 |
| `isActive` | `boolean` | 当前 tab 是否活跃 |
| `theme` | `string` | 当前主题 |
| `language` | `string` | 当前语言 |
| `onClose` | `() => void` | 关闭回调 |

### 外部插件方式（模板驱动）

通过 `termcat.ui.registerPanel()` 注册模板驱动面板。

```typescript
termcat.ui.registerPanel({
  id: 'my-panel',
  title: 'My Panel',
  icon: 'zap',                  // lucide-react 图标名
  slot: 'sidebar-left',         // 'sidebar-left' | 'sidebar-right' | 'bottom-panel'
  defaultSize: 280,
  defaultVisible: true,
  priority: 20,
});

// 推送面板数据（模板描述）
termcat.ui.setPanelData('my-panel', [
  {
    id: 'header',
    template: 'header',
    data: { title: 'My Panel', subtitle: 'v1.0', icon: 'zap' },
  },
  {
    id: 'info',
    template: 'key-value',
    data: { pairs: [{ key: 'Status', value: 'Running' }] },
  },
]);

// 局部更新某个 section
termcat.ui.updateSection('my-panel', 'info', {
  pairs: [{ key: 'Status', value: 'Stopped', color: 'danger' }],
});
```

## 2. 底部面板（Bottom Panel）

仅内置插件支持，通过 `context.registerBottomPanel()` 注册到底部 Tab 区域。

```typescript
import { Terminal } from 'lucide-react';

context.registerBottomPanel({
  id: 'my-bottom-panel',
  title: 'My Panel',                          // fallback 标题
  getLocalizedTitle: (lang) => locale[lang],   // 多语言标题
  icon: Terminal,                              // lucide-react 组件
  priority: 30,                                // 越小越靠前
  component: MyBottomPanel,
});
```

**组件接收的 Props（`BottomPanelProps`）：**

| Prop | 类型 | 说明 |
|------|------|------|
| `connectionId` | `string \| null` | SSH 连接 ID |
| `fsHandler` | `IFsHandler` | 文件系统操作能力 |
| `theme` | `string` | 当前主题 |
| `isVisible` | `boolean` | 当前 tab 是否可见 |

## 3. 工具栏按钮（Toolbar Toggle）

内置插件可注册工具栏按钮，用于控制面板的显示/隐藏。

```typescript
import { LayoutPanelLeft } from 'lucide-react';

context.registerToolbarToggle({
  panelId: 'my-sidebar',        // 关联的面板 ID
  icon: LayoutPanelLeft,         // React 组件
  tooltip: '切换面板',
  priority: 10,                  // 越小越靠前
});
```

## 4. 状态栏（Status Bar）

外部插件通过 `termcat.ui.registerStatusBarItem()` 在底部状态栏添加信息。

```typescript
const item = termcat.ui.registerStatusBarItem({
  id: 'my-status',
  text: 'Connected',
  tooltip: 'SSH connection active',
  icon: 'wifi',
  position: 'left',     // 'left' | 'right'
  priority: 10,
  onClick: () => { /* 点击处理 */ },
});

// 动态更新
termcat.ui.updateStatusBarItem('my-status', { text: 'Disconnected' });
```

## 8. 模板驱动面板 — 可用模板

模板驱动面板通过 JSON 数据描述 UI，由宿主预定义的模板组件负责渲染。

### P0 基础模板

| 模板 | 说明 | 关键数据字段 |
|------|------|-------------|
| `header` | 标题栏 | `title`, `subtitle`, `icon`, `badge`, `actions` |
| `key-value` | 键值对列表 | `pairs[]`, `layout` ('vertical'/'horizontal'/'grid') |
| `metric-bars` | 进度条指标 | `items[]` (label, value, max, unit, color) |
| `table` | 数据表格 | `columns[]`, `rows[]`, `maxVisibleRows`, `virtualScroll` |
| `bar-chart` | 柱状图/折线图 | `series[]`, `maxPoints`, `height`, `yUnit` |
| `divider` | 分割线 | `label` |

### P1 扩展模板

| 模板 | 说明 | 关键数据字段 |
|------|------|-------------|
| `list` | 列表 | `items[]` (label, icon, description, badge, actions) |
| `tree-view` | 树形视图 | `nodes[]` (嵌套 children) |
| `log-stream` | 日志流 | `lines[]` (timestamp, level, message), `autoScroll` |
| `button-group` | 按钮组 | `buttons[]` (label, icon, color, variant) |
| `text` | 文本块 | `content`, `format` ('plain'/'code'/'pre') |

### P2 高级模板

| 模板 | 说明 |
|------|------|
| `metric-ring` | 环形仪表盘 |
| `sparkline` | 迷你趋势线 |
| `area-chart` | 面积图 |
| `columns` | 多列布局（嵌套 SectionDescriptor） |
| `grid` | 网格布局 |
| `progress` | 进度指示器 |
| `tabs` | 标签页（每个 tab 含 sections） |
| `form` | 简单表单 |
| `status-bar` | 状态栏条目 |
| `notification` | 通知列表 |

### 模板变体

每个模板支持三种变体：`default` | `compact`（紧凑）| `card`（卡片）。

### 主题颜色

模板中的 `color` 字段支持以下预定义主题色：

| 颜色 | 映射 |
|------|------|
| `primary` | indigo |
| `success` | emerald |
| `warning` | orange |
| `danger` | red |
| `info` | cyan |
| `muted` | slate |

也支持自定义 hex/rgb 值。

## 11. 事件系统

### 内置插件事件

```typescript
// 监听连接变化
context.onConnectionChange((info) => {
  if (info) {
    console.log('Connected:', info.hostname);
  }
});

// 跨插件事件
context.emitEvent('myPlugin:dataReady', { data: 'xxx' });
```

### 外部插件事件

```typescript
// 终端事件
termcat.terminal.onDidOpenTerminal((terminal) => { /* ... */ });
termcat.terminal.onDidCloseTerminal((terminal) => { /* ... */ });
termcat.terminal.onData(sessionId, (data) => { /* ... */ });

// SSH 事件
termcat.ssh.onDidConnect((info) => { /* ... */ });
termcat.ssh.onDidDisconnect((info) => { /* ... */ });

// 文件事件
termcat.files.onDidOpenFile((file) => { /* ... */ });
termcat.files.onDidSaveFile((file) => { /* ... */ });

// AI 事件
termcat.ai.onMessage((message) => { /* ... */ });
termcat.ai.onTaskStateChange((state) => { /* ... */ });

// 监控事件
termcat.monitor.onMetricsUpdate((metrics) => { /* ... */ });
```

## 12. 声明式扩展（package.json contributes）

外部插件可在 `package.json` 的 `termcat.contributes` 中声明式注册：

```json
{
  "termcat": {
    "contributes": {
      "commands": [
        { "id": "myPlugin.doSomething", "title": "Do Something", "icon": "play" }
      ],
      "menus": {
        "terminal.toolbar": [
          { "command": "myPlugin.doSomething", "when": "ssh.connected" }
        ]
      },
      "sidebar": {
        "id": "myPlugin.sidebar",
        "title": "My Plugin",
        "icon": "puzzle",
        "position": "bottom"
      },
      "settings": {
        "autoRefresh": {
          "type": "boolean",
          "default": true,
          "description": "自动刷新数据"
        }
      }
    }
  }
}
```
